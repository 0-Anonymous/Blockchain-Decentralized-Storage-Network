# Blockchain Decentralized Storage Network

A blockchain-based decentralized file storage system developed as a Final Year B.E. Computer Engineering project.

## Features

- Secure User Authentication
- AES-256 File Encryption
- File Chunking and Distribution
- Blockchain Transaction Logging
- Merkle Tree Verification
- Peer-to-Peer File Sharing
- UDP Hole Punching
- Azure Relay Backup
- Multi-device File Access
- File Manifest Synchronization

## Technologies Used

- C#
- .NET
- WPF
- ASP.NET Core Web API
- Azure App Service
- Azure SQL Database
- Blockchain
- UDP Networking
- AES-256 Encryption
